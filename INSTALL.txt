# 企业级交换路由自动配置系统 - 安装说明

## 依赖安装

由于Python 3.8的网络兼容性问题，请按照以下步骤安装依赖：

### 方法1：使用安装脚本（推荐）

双击运行 `install.bat` 文件

### 方法2：手动安装

打开命令提示符，执行：

```bash
pip install --upgrade pip
pip install paramiko requests
```

### 方法3：如果上述方法失败

1. 下载离线whl文件：
   - paramiko: https://pypi.org/project/paramiko/#files
   - requests: https://pypi.org/project/requests/#files

2. 离线安装：
```bash
pip install paramiko-3.x.x-py3-none-any.whl
pip install requests-2.x.x-py3-none-any.whl
```

### 方法4：使用conda

如果你使用Anaconda：
```bash
conda install paramiko requests
```

## 运行程序

安装依赖后，运行：
```bash
python main.py
```

## 系统要求

- Python 3.8+
- Windows/Linux/macOS
- 网络连接（用于调用AI API）
