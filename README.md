# 配网通 V1.0 — 企业级交换路由自动配置系统

## 产品概述

**配网通**是一款面向企业网络运维人员的智能化网络管理与配置工具。系统深度融合 AI 大模型技术与传统网络诊断能力，提供从设备配置、网络计算到运维监控的一站式解决方案，大幅提升网络工程师的工作效率。

**制作人**：林鹏  
**版本号**：V1.0  
**发布日期**：2026年

## 核心功能

### 一、AI 智能配置

利用先进的大语言模型（支持 OpenAI、Claude、通义千问、文心一言、豆包、DeepSeek、Qwen 及自定义 API），根据用户自然语言描述的配置需求，自动生成交换机、路由器等网络设备的专业配置命令。

- **设备选型**：内置主流厂商（华为、H3C、锐捷、思科等）设备型号数据库，支持厂商 → 类型 → 型号三级联动选择
- **智能生成**：输入配置需求（如"配置 VLAN 10，端口 1-10 加入 VLAN 10"），AI 自动生成对应的 CLI 命令
- **结果分析**：对 AI 返回的配置结果进行结构化展示，便于审核和复制
- **配置历史**：自动保存历史配置记录，支持一键恢复
- **API 持久化**：首次填写 API Key 后自动保存，下次启动无需重复输入，支持一键清除

### 二、网络计算工具

提供网络工程师日常工作中常用的计算工具：

- **IP 子网计算器**：输入 IP 地址和子网掩码，自动计算网络地址、广播地址、可用主机范围、主机数量等
- **VLSM 计算器**：支持可变长子网掩码划分，根据需求自动分配最优子网
- **MAC 地址计算**：MAC 地址格式转换与计算
- **网络掩码转换**：CIDR 前缀长度与子网掩码互转

### 三、网络运维工具

#### 3.1 仪表板

实时展示本机网络状态概览：

- **内网 IP**：自动获取本机局域网 IP 地址（支持复制）
- **外网 IP**：通过多 API 备选获取公网 IP（支持复制）
- **MAC 地址**：显示本机网卡 MAC 地址（支持复制）
- **CPU 占用率**：实时监控系统 CPU 使用率
- **内存占用率**：实时监控系统内存使用情况
- **网络丢包率**：快速检测网络连通质量
- **系统运行时间**：显示系统已运行时长

#### 3.2 网络健康检查

一键执行多项网络诊断：

- Ping 网关检测
- Ping 外网（DNS）检测
- DNS 解析测试
- 路由追踪（Traceroute）
- 端口连通性检测

#### 3.3 外网测速

测试本机互联网连接速度：

- **下载速度**：使用 Cloudflare、CacheFly 等多源测速
- **上传速度**：测试上行带宽
- 实时显示测速进度

#### 3.4 内网测速

测试局域网内两台设备之间的传输速度：

- 支持 TCP / UDP 协议
- 可自定义目标 IP 和端口
- 实时显示传输速率和平均速率

#### 3.5 路由追踪（Traceroute）

追踪数据包从本机到目标主机的完整路径：

- 显示每一跳的 IP 地址和延迟
- 支持设置最大跳数
- 提供**停止按钮**，可随时中断追踪

#### 3.6 端口扫描

扫描目标主机的开放端口：

- 支持自定义 IP 范围和端口范围
- 支持 TCP / UDP 协议
- 批量扫描，结果显示在列表中
- 提供**停止按钮**，可随时中断扫描

#### 3.7 主机发现

自动发现局域网内的在线设备：

- **三阶段扫描**：Ping 扫描 → SYN 端口扫描 → ARP/NBT 信息收集
- 自动获取本机网段作为扫描范围
- 显示设备 IP、MAC 地址、设备名称、状态
- 通过 ARP 表解析 MAC 地址，自动识别厂商（内置 39,000+ OUI 记录）
- 提供**停止按钮**，可随时中断扫描

#### 3.8 摄像头扫描

专门针对网络摄像头的发现工具：

- 自动获取本机网段
- 扫描常见摄像头端口（554/RTSP、80/HTTP、8000、8080 等）
- 识别海康威视、大华、宇视等主流品牌
- 提供**停止按钮**，可随时中断扫描

#### 3.9 外网 IP 检测

快速检测本机公网 IP 地址：

- 多 API 备选（ipify.org、ifconfig.me、seeip.org）
- 显示 IP 地址、所在城市、运营商信息

#### 3.10 DHCP 检测

检测本机 DHCP 配置信息：

- 自动识别所有网络适配器（有线、无线、蓝牙）
- 显示 DHCP 服务器地址、IP 地址、子网掩码、默认网关、DNS 服务器
- 区分 DHCP 自动获取和静态 IP 配置

#### 3.11 MAC 厂商查询

根据 MAC 地址查询设备厂商：

- 内置 39,000+ 条 OUI（组织唯一标识符）记录
- 覆盖华为、H3C、锐捷、思科、海康威视、大华、宇视等主流厂商
- 支持手动输入 MAC 地址查询

#### 3.12 网络接口信息

显示本机所有网络接口的详细信息：

- 接口名称、IP 地址、子网掩码、MAC 地址
- 发送/接收数据包统计
- 接口状态（UP/DOWN）

## 技术特点

| 特性 | 说明 |
|------|------|
| **AI 驱动** | 支持 8 种主流 AI 模型，智能生成网络设备配置命令 |
| **单文件部署** | 打包为独立 EXE 文件，无需安装 Python 环境，双击即用 |
| **多线程并发** | 网络扫描、测速等耗时操作均在后台线程执行，界面不卡顿 |
| **自动网段识别** | 主机发现、摄像头扫描等自动获取本机局域网网段 |
| **MAC 厂商识别** | 内置完整 OUI 数据库，自动识别设备厂商 |
| **配置持久化** | AI API 配置、命令历史自动保存，重启不丢失 |
| **可复制信息** | 仪表板 IP、MAC 等信息支持一键复制 |
| **可控扫描** | 所有扫描任务均提供停止按钮，随时中断 |

## 系统要求

- **操作系统**：Windows 7 / 8 / 10 / 11（64 位）
- **运行环境**：无需安装任何依赖，开箱即用
- **网络要求**：AI 配置功能需要互联网连接；网络运维工具需要局域网连接

## 使用说明

1. **AI 智能配置**：选择设备厂商和型号 → 输入配置需求 → 点击"生成命令" → 复制命令到终端执行
2. **网络运维工具**：切换到"网络运维工具"标签页，选择对应功能模块即可使用
3. **API 配置**：首次使用 AI 功能需填写 API Key，点击"保存"后下次自动加载
